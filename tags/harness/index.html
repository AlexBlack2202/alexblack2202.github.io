Ngôn ngữ tự nhiên rất tiện, nhưng cũng rất mơ hồ. Khi AI agent chỉ suy luận bằng chữ, nó có thể nói nghe rất hợp lý nhưng vẫn sai. Hướng Code as Agent Harness đề xuất một cách nhìn...
